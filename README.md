# Bot Logistic

Мощный Telegram-инструмент для логистических операций, мониторинга грузов и массовой рассылки сообщений.

## Основные возможности

- **Мониторинг грузов**: Автоматическое сканирование сообщений в группах на наличие маршрутов (например, "Ташкент - Москва") и контактных данных.
- **Userbot Manager**: Управление множеством аккаунтов (юзерботов) для мониторинга и рассылки.
- **Массовая рассылка**: Планировщик рассылок по группам с гибким выбором интервалов (от 1 до 6 часов).
- **Управление папками**: Автоматическое вступление в группы по ссылкам-приглашениям (Addlists) и распределение их по категориям (Папки 1-4).
- **Админ-панель**: Интуитивно понятный интерфейс через Telegram-бота для управления доступами и аккаунтами.

## Установка и запуск

1. Склонируйте репозиторий.
2. Создайте виртуальное окружение: `python -m venv venv`.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Настройте файл `.env` (API_ID, API_HASH, BOT_TOKEN).
5. Запустите основной скрипт: `python3 main.py`.

## Технологии

- **Python 3.10+**
- **Aiogram 3.x** (Admin Bot)
- **Telethon** (Userbots)
- **SQLite (aiosqlite)** (База данных)
- **APScheduler** (Планировщик задач)
